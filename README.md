# PocketLedger / 随心记账本

一个面向大学生的本地优先、开源、桌面端记账工具。

## 当前能力

- 支持支出、收入、转账三类交易
- 支持账户管理：微信、支付宝、银行卡、现金
- 支持分类管理和自定义分类
- 首页直接看本月生活费、已花、剩余、收入、支出、结余
- 支持账单筛选、搜索、最近交易回看
- 支持分类统计、账户支出分布、趋势图
- 支持 JSON / CSV 导出，支持 JSON 导入
- 支持账户余额校准和校准日志
- 默认本地存储，无需注册登录

## 技术栈

- React
- Vite
- Electron
- 本地 JSON 文件存储

## 完全由Codex开发

本工具完全由 Codex 开发，从写PRD，设计UI，到编写代码，提交至Github，全部都是通过Codex完成
